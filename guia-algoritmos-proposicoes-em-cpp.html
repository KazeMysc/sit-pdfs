Aplicacação em C++

Proposições Lógicas em C++

Como vimos anteriormente, proposições lógicas são frases que podem ser verdadeiras ou falsas.

"Mas como poderíamos aplicar isso na programação?" - você deve estar se perguntando.

Na verdade, é bem simples, o que vai mudar será a linguagem usada, um pouco da estrutura e o idioma, que será em Inglês.

Exemplo
//Iniciamos o código como sempre
#include <iostream>
using namespace std;
int main() {
//Declaramos variáveis do tipo inteiro
int numero1 = 2;
int numero2 = 2;
      
//Agora comparamos de forma simples
if ( 2 == 2 ) { cout << "São iguais!" << endl;}
return 0;
}

No código, incluímos a biblioteca de entrada e saída de dados iostream:

Exemplo
//Iniciamos o código como sempre
#include <iostream>

Depois, escrevemos no código using namespace std para que não precisemos escrever std:: antes dos nomes dos elementos da biblioteca padrão, como o próprio cout:

Exemplo
//Iniciamos o código como sempre
#include <iostream>
using namespace std;

Então criamos nossa função principal int main() para que nós possamos começar a programar:

Exemplo
//Iniciamos o código como sempre 
#include <iostream>
using namespace std;
int main() {
return 0;
}

Dentro do int main, declaramos duas variáveis do tipo inteiro que armazenarão valores numéricos, nesse caso, 2:

Exemplo
//Iniciamos o código como sempre 
#include <iostream>
using namespace std;
int main() {
//Nossas variáveis do tipo inteiro
int numero1 = 2;
int numero2 = 2;
return 0;
}

Após isso, codificamos uma declaração if, que funciona exatamente da mesma forma que em pseudocódigo, que abordaremos no próximo capítulo.

Exemplo
//Iniciamos o código como sempre 
#include <iostream>
using namespace std;
int main() {
//Nossas variáveis do tipo inteiro
int numero1 = 2;
int numero2 = 2;

//Nossa declaração if (se)
if ( 2 == 2 ) {cout << "São iguais!" << endl;}
return 0;
}

Vamos analisar juntos:

Declaramos um if que compara se 2 é igual a 2, caso seja, ele vai "pensar": "Bem, 2 é igual a 2, então é verdade! Eles são iguais! Nesse caso, vou exibir no console que são iguais"

Isso que codificamos:

Exemplo
//Nossa declaração if (se)
if ( 2 == 2 ) {cout << "São iguais!" << endl;}
return 0;
}

Como a condição foi cumprida, ou seja, como 2 é igual a 2, será exibido no seu console a seguinte mensagem:

Essa é a saída de dados, ou em inglês, output.

Em Pseudocódigo

Em pseudocódigo, o código ficaria da seguinte maneira:

Exemplo
Inicio

Inteiro numero1 ← 2 //int numero1 = 2
Inteiro numero2 ← 2 //int numero2 = 2

Se numero1 == numero 2 então  
   Escreva "São iguais!"

//O mesmo que nosso: if ( 2 == 2 )
